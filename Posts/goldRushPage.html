Gold Rush

I created Gold Rush as a game jam project to experiment with multiplayer networking and random generation. The gameplay involves racing other players to mine as much gold as possible within the time limit. I used a tile system and Perlin noise to generate a unique cave system each game. I programmed a movement system that works with the tile system by gently aligning the players to the tile grid. This prevents the players from getting snagged on tiles so that the fast movement and game pacing are never broken.

I decided to make a website that acts as the game controller so that players can join a local game using their computer or phone. The website lets players input a name and connection code to connect to the game server, then use the arrow buttons on the website or their keyboard arrow keys to control their character. The website sends WebSocket messages that are received by a TCP server and used to move that player's miner.
