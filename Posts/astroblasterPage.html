Astroblaster

Astroblaster is an endless arcade shooter with periodic upgrade choices and new enemy spaceships that appear as the difficulty increases. I developed Astroblaster using Unity and C# to further improve my skill with these technologies. I handled all of the design and programming myself and experienced the process of designing and implementing gameplay features from start to finish. When creating the UI, I extended Unityâ€™s default text functionally to create curved text. The randomly generated background was programmed to avoid patterns and maintain consistent spacing among its elements. I also optimized the game for the large number of ships, projectiles, and background elements that could be in play at any given moment.

To implement the circular UI, I wrote a shader to draw circles with adjustable space for the text. My curved text implementation improved upon this GitHub project. The original code moved text vertices individually, which resulted in some letters being noticeably warped, so I modified it to transform the center point of each text quad, then offset the vertices in local space. This maintained the normal spacing of the vertices and prevented deformation.

The circle drawing shader I programmed uses adjustable width and gaps to easily fit any text.

Deformation of the letters, especially the C and stem of the T, can be seen on the left. My implementation transforms the center of each letter to avoid this effect, as seen on the right.
